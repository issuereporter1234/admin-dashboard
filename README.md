# Admin Dashboard

A responsive admin dashboard layout built with CSS Grid.

🔗 [Live Demo](https://issuereporter1234.github.io/admin-dashboard/)

## Features

- Full page layout using CSS Grid
- Sidebar navigation
- Header with search and user info
- Project cards and activity sections
- Links to other projects via the eye icons on each project card

## Tech Stack

- HTML
- CSS (Grid)

## How to Use

Clone the repo and open `index.html` in your browser — no installation required.

```bash
git clone https://github.com/issuereporter1234/admin-dashboard
cd admin-dashboard
```
